# Salesforce Quiz Leaderboard Assignment

## Overview
This project is created for Bajaj Finserv Health internship assignment.

## Implementation
- Apex class created
- Logic includes:
  - API polling (10 times)
  - Deduplication using (roundId + participant)
  - Score aggregation using Map
  - Leaderboard generation

## Note
Due to Salesforce platform limitations, delay between API calls is not implemented.
